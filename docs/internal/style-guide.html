Usage Guidelines

Best Practices

  • ✓ Always use design tokens from the system, never hardcode values
  • ✓ Maintain typography hierarchy (H1 > H2 > H3 > Body)
  • ✓ Use spacing scale consistently (8px base unit)
  • ✓ Keep animations subtle and purposeful
  • ✓ Ensure WCAG AA contrast ratios for accessibility

Source of Truth

All design tokens are defined in src/design-system/presentation-theme.ts

To modify any design token, edit that file and regenerate presentations.

See docs/design-system-presentation.md for complete documentation.
