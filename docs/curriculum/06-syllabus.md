## Weekly Schedule

### Week 1: AI Foundations

**Theme**: Understanding What AI Actually Is (And Isn't)

| Day | Activity | Time | Deliverable |
|-----|----------|------|-------------|
| 1-2 | Read Module 1.1: Economics of Intelligence | 1.5 hr | Notes on token economics |
| 3-4 | Read Module 1.2: Context and Memory | 1.5 hr | Notes on context windows |
| 5-6 | Start Lab 1: Persona Stress Test | 2 hr | — |
| 7 | Complete Lab 1 | 1 hr | Persona comparison table |

## Weekly Office Hours / Check-ins

For self-paced learners, schedule weekly self-check-ins:

### Self-Assessment Questions

**Week 1-2 (Literacy)**:
- Can I explain tokens, context windows, and RAG to a non-technical person?
- Can I get meaningfully different outputs by changing AI personas?
- Do I understand why different models exist?

**Week 3-4 (Workflows)**:
- Can I break a complex task into atomic operations?
- Can I write process documentation that requires no interpretation?
- Do I know where to put human checkpoints?

**Week 5-6 (Implementation)**:
- Can I build a working automation from scratch?
- Do I understand triggers, actions, and data flow?
- Can I handle common error scenarios?

**Week 7-8 (Strategy)**:
- Can I calculate AI ROI with specific numbers?
- Can I identify AI-specific risks and design mitigations?
- Can I write a recommendation memo that would convince a skeptical executive?

| **Good** | Meets all requirements, solid execution, minor improvements possible |

### Portfolio-Readiness Criteria

A portfolio-ready deliverable:
- Solves a real problem (not just completes an exercise)
- Demonstrates specific skills clearly
- Could be explained in a job interview
- Shows iteration and improvement
- Is professionally presented

## Frequently Asked Questions

**Q: What if I fall behind schedule?**
A: The curriculum is designed to be flexible. Each phase builds on previous ones, but you can adjust timing. The key milestones are the phase deliverables.

**Q: Do I need to complete labs in order?**
A: Yes, labs build on each other. Phase 3 labs assume Phase 2 knowledge, etc.

**Q: Can I use different AI tools than recommended?**
A: Yes, the concepts transfer. ChatGPT, Claude, and Gemini all work for the labs. For automation, Make or n8n are alternatives to Zapier.

**Q: How do I know if my work is good enough?**
A: Each lab includes evaluation criteria. Ask: "Could I explain this in a job interview? Would I be proud to show this to a potential employer?"

**Q: What if I get stuck?**
A: Re-read the relevant module, try a simpler version of the task first, or use AI to help debug your approach (but don't let it do the work for you).
